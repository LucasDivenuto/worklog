# Rustikas Landing

Sitio institucional estatico para Rustikas.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El build queda en `dist/` y puede desplegarse copiando su contenido a:

```text
/var/www/rustikas-site
```

## Configuracion

El telefono de WhatsApp esta centralizado en `src/siteData.js`.
